Summary: Latitude vs Weather Type
The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the Open Weather Map API to assemble a dataset on over 500 cities. After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs. latitude. Factors we looked at include: Temperature, Cloudiness, Wind Speed, and Humidity. This site provides the source data and visualations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
